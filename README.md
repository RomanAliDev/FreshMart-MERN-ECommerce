# FreshMart - Full Stack MERN Grocery Store

FreshMart is a full-stack grocery e-commerce web application developed using the MERN stack. The platform allows users to browse products, manage carts, save delivery addresses, and place orders, while administrators can manage products through a dedicated admin dashboard.

## Features

* User Authentication (Login & Signup)
* Role-Based Access Control (Admin/User)
* Product Management (Add, Update, Delete)
* Shopping Cart Functionality
* Checkout & Address Management
* Order Placement System
* Admin Dashboard
* Responsive Design
* RESTful API Integration

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* React Router
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication


## Future Enhancements

* Online Payment Integration
* Order Tracking
* Product Reviews & Ratings
* AI-Based Product Recommendations
